# AlphaNexus Cloud

## Projeto de Integração Multi-Cloud Azure + AWS

### Objetivo

Implementar uma arquitetura Multi-Cloud conectando Microsoft Azure e Amazon Web Services (AWS) através de uma VPN Site-to-Site, permitindo comunicação segura entre máquinas virtuais utilizando apenas IPs privados.
